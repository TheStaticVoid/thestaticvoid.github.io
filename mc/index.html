Welcome!

What's the point of my Minecraft server having a website? Well some of the datapacks included on the server have some odd or complex recipes that I figured it would be much easier to have a page to look them up, rather than having to visit each of their pages individually. The plugins used in the server have a LOT of commands, so I felt it would be best to have a page to help get accustomed to new and different plugins.
